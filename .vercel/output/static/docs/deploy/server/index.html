服务器托管

Node 预设是 Nuxt 和 Nuxt Content 的默认预设。它用于在 Node.js 上构建和运行 Nuxt 应用程序。

什么是 Node.js 预设？
环境要求
使用 Node.js 预设构建

什么是 Node.js 预设？

Node 预设是 Nuxt 的默认预设，当构建您的项目时，Nuxt 将输出一个 Node.js 服务器，您可以使用 node .output/server/index.mjs 运行它。

环境要求

如果您使用默认的 better-sqlite3 模块来操作 sqlite 数据库， 那么您必须部署到 Glibc 版本高于 2.29 的操作系统，例如 Debian 11、Ubuntu 20.04。

您可以使用 ldd --version 来检查 Glibc 版本。查看 issue #3248 了解更多详情。

使用 Node.js 预设构建

使用 Nuxt 构建命令构建项目：

Terminal
nuxi build

当使用 Node 服务器预设运行 nuxi build 时，结果将是一个启动即用型 Node 服务器的入口点。

Terminal
$ node .output/server/index.mjs
Listening on http://localhost:3000
SQLite 数据库将在服务器端启动服务器时加载，也会在浏览器中加载以用于客户端导航或操作。
