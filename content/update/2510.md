### 📦 更新方式

#### Docker 部署
1. git 拉取最新代码：`git reset --hard HEAD && git pull`
2. 重启 Docker 中 智言万象 下的 Nodejs 容器：`docker restart 智言万象-nodejs`

#### 手动部署
本地有 pnpm 和 git 则先执行 `pnpm sync-env`，然后执行：
`pnpm bd:update-git && pnpm restart`
或者执行：
`git reset --hard HEAD && git pull && pnpm predeploy && pnpm restart`
