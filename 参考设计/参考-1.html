常见问题解答

你需要知道的一切。

概述

“终身访问”到底是什么意思？

购买Tailwind Plus许可证为一次性购买，无需定期订阅。购买Tailwind Plus许可证后，你可以永久访问Tailwind Plus中的所有内容。

“免费更新”包括哪些内容？

购买Tailwind Plus许可证后，您可以免费使用目前可用的所有组件包和网站模板，以及我们未来发布的任何额外组件包或模板。

这意味着如果我们添加全新的组件包，比如“新闻”，或者设计新的社交媒体网站模板，这些产品的访问权限都会包含在你最初的购买中。

想了解过去的更新情况，可以查看我们的更新日志。

你们是否提供学生或购买力平价折扣？

对于价格有问题的情况，我们提供折扣的限制性执照，您可以购买以下条件：

  • 你是一名纯粹出于教育目的使用 Tailwind Plus 的学生，或者
  • 您是个人，使用 Tailwind Plus 进行商业项目，且您来自各类的年个人收入不超过 30,000 美元

如需购买折扣限制许可，请发送邮件至 support@tailwindcss.com。

兼容性

Figma、Sketch 或 Adobe XD 文件包含在内吗？

不，Figma、Sketch 或 Adobe XD 等工具的设计资源不包含在内。

我们自身设计和开发流程中并不产出高质量的设计成果，因此积累这些额外资源意味着我们无法花太多时间在代码中创建新的组件和模板，而我们认为这正是我们能提供最大价值的地方。

使用的是哪种 JS 框架？

我们的网站模板是用Next.js构建的，所以所有标记都是用React写的。

我们的组件提供三种格式：React、Vue 和原生 HTML。

当需要交互性（比如下拉菜单、对话框等）时，我们用Headless UI来处理React和Vue的例子，用Elements来处理HTML示例。这两个库都提供了无样式组件，我们设计时能完美集成到 Tailwind CSS。

想更好地了解实际操作效果，请查看我们的文档。

Tailwind CSS 使用的是哪个版本？

Tailwind Plus 中的一切都是为最新版本的 Tailwind CSS 设计和开发的，目前是 Tailwind CSS v4.2。

支持哪些浏览器？

Tailwind Plus 中的组件和模板设计用于所有主流浏览器的最新稳定版本，包括 Chrome、Firefox、Safari 和 Edge。

授权

我需要为我参与的每个项目购买许可吗？

与大多数其他模板/主题不同，你不必每次想在新项目中使用时都购买新的 Tailwind Plus 许可证。

只要你建造的内容符合许可，你就可以建造任意多的网站，而无需购买额外的许可。

欲了解更多信息和示例，请阅读我们的许可条款。

我可以用Tailwind Plus来做客户项目吗？

是的！只要你构建的是为单一客户开发的定制网站，你完全可以用Tailwind Plus的组件和模板来帮助构建。

唯一不能做的是用Tailwind Plus搭建一个网站或模板，然后转售给多个客户。

欲了解更多信息和示例，请阅读我们的许可条款。

我可以用Tailwind Plus来做自己的商业项目吗？

绝对是！你的许可证允许你自行开发任意数量的项目，无论是简单的公共网站还是终端用户需要付费访问的SaaS应用。

只要你构建的不是网站建设工具或其他客户可以用来创建自己网站的工具，那就没问题。

欲了解更多信息和示例，请阅读我们的许可条款。

我可以在开源项目中使用Tailwind Plus吗？

没错！只要你构建的是某种实际网站，而不是衍生组件库、主题构建器或其他明确目的是重新打包和重新分发组件的产品，那么该项目开源完全没问题。

如需更多信息及什么可以、什么不可以的例子，请阅读我们的许可。

我可以用Tailwind Plus制作的模板/主题出售吗？

不，你不能用Tailwind Plus来创建衍生产品，比如主题、UI套件、页面构建器，或者任何你会重新打包和分发组件或模板，供别人用来构建他们自己的网站的东西。

如需更多信息及什么可以、什么不可以的例子，请阅读我们的许可。

我可以以后升级到Team授权吗？

没错！如果你是独立开发者，可以先用普通许可证，之后如果有其他开发者加入团队，再升级到团队许可证。

登录后，在“账户”菜单下有一个“升级为团队许可”选项。

我可以购买包含超过25个座位的球队执照吗？

一般来说，如果你需要超过25个座位，我们建议购买多个球队执照——为公司中每个需要使用权的团队各申请一个。

例如，如果你经营一家电商业务，你可能会为店铺团队购买一个团队许可证，然后再为市场团队购买另一个团队许可证，接着为数据科学团队购买另一个团队许可证，依此类推。

如果你需要超过25个座位，因为你是大型代理机构或其他类型的公司，没有真正划分成独立部门/团队，请联系我们 support@tailwindcss.com 获取大型团队的定价。

支持

你们提供技术支持吗？

Tailwind Plus 是一款自助产品，这意味着虽然我们提供客户支持，比如账户管理和授权相关问题，但期望客户具备 Tailwind CSS、HTML、React 和 Vue 的必要知识，以便成功使用该产品。

由于每个项目都不同，且独立编写的代码交互方式复杂且耗时，我们不提供技术支持或咨询服务。

你们的退款政策是什么？

如果您对购买不满意，请在30天内发送邮件至 support@tailwindcss.com，我们将全额退款，毫无疑问。
